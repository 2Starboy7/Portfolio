    Palmprint Recognition Using Multiview Learning

    Palmprint recognition is popular for security authentication due to its unique features like local direction, wrinkles, and texture. However, factors such as noise, rotations, and shadows can degrade performance. To improve recognition, we use Multiview Feature Learning (MFL), which leverages complementary features from multiple views. MFL, proven effective in applications like image retrieval and speech recognition, can achieve better performance by incorporating more comprehensive information than single-view methods.

    Intelligent Powerline Management System Using IOT

    Our powerline management system enhances the efficiency and safety of power transmission by automatically shutting down electricity in broken power lines, preventing accidents. Unlike the conventional manual method, our system uses an NRf24L01 transceiver to send signals from a Nano Ch340 to another receiver connected with a Nano and relay, ensuring immediate power shutdown. This project aims to improve the reliability and response time of power transmission infrastructure.
